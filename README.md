# Car Manager — iOS Vehicle Service Tracker
A native iOS application built with SwiftUI and Core Data for tracking vehicle maintenance and service events. Users can manage multiple vehicles, log service events with location data, and visualize expenses through charts.

## Technologies:

- Swift / SwiftUI - declarative UI framework
- Core Data - local persistent storage
- MapKit - location picking and map display
- Charts - pie chart for expense breakdown by category

## Features:

- Add and manage multiple vehicles (name, type, fuel type, photo)
- Log service events (refueling, maintenance, repairs, etc.)
- Attach GPS location to each event via interactive map
- View expense statistics with charts per vehicle
- Native iOS look & feel with NavigationView and List
